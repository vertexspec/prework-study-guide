HTML

  • HTML Stands for Hypertext Markup Language (HTML), to give the content structure.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS Stands for Cascading Style Sheets (CSS), to give the content style.
  • Link the CSS file to the HTML file by adding in the code before the title and after all the meta code lines
  • Essentially you need to use linking code between the HTML and CSS file is how it implements the style attributes
  • Also adding sections helps with formatting between the HTML file and the CSS file
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status:checks what branch we are currently on
  • git checkout -b branch-name:creates a new branch and switches to it
  • git add -A:adds a file as it looks now to your next commit (stage)
  • git commit -m "Descriptive Message": Commits your staged content as a new commit snapshot

Javascript

  • Javascript language creates interactivity with a website
  • The Language was invented by Brendan Eich
  • It is to be linked within the body element in HTML
  • the code "console.log" allows the use of Chrome DevTools now in the live server
  • "Console.log" was created to make it easier for developers to test their code as they can use it to debug and test their code as soon as they write it
  • Control Flow is the order in which a computer executes code in a script. It runs from the top of the page to the bottom of the page.
  • It can be controlled through the use of Conditional Statements and loops
